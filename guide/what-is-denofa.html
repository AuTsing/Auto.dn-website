Auto.dn 是什么？ ​

Auto.dn 是一个基于 Deno(类似于 Node.js 的 JavaScript 和 TypeScript 运行时) 的自动化平台。目前已支持安卓平台。

Auto.dn 基于 Deno，因此你可以使用任何 Deno 内置的所有 API，因为 Deno 兼容 npm 生态，因此你还可以获得丰富的 npm 社区生态支持。

在此基础上，Auto.dn 还提供丰富的安卓 API，你可以非常轻松的操作安卓平台。

功能 ​

  • ✅ 运行工程
  • ✅ 管理工程
  • ✅ 查看日志
  • ✅ 运行 WS 服务器
  • ✅ 运行 FTP 服务器
